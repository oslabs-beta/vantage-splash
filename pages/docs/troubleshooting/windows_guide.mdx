# Troubleshooting: Running Vantage on Windows

## Puppeteer

Vantage uses puppeteer under the hood to launch a browser for google lighthouse to run. Check out the puppeteer Troubleshooting docs for more help.

https://github.com/puppeteer/puppeteer/blob/main/docs/troubleshooting.md

## Error Log

Vantage will add any errors that occure to the error log in the vantage folder. Take a look there when trying to debug.

# Key Lighthouse Metrics

Below is a summary of each of the key top level metric categories:
* **Performance** | Identifies areas that may lead to slow performance or reduced responsiveness, such as large images.
* **Accessibility** | Analyzes performance against accessibility best practices, including compatibility with assistive technology. 
* **Best Practices** | Recommendations against the best practices in modern web development.
* **SEO** | Highlights potential improvements to improve search engine performance.
* **Progressive Web App** | Key metrics that ensure you have a high-performing progressive web app.

In the vantage_report.html file, you can select two points on the graph, then hover over the list of recommendations to see further detail on each.

For a detailed analysis of how Google Lighthouse computes your project's score, we recommend this <a href="https://web.dev/performance-scoring/">article</a>.
